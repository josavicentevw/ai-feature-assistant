# Skill: Codebase Assessment

## Goal
Identify existing components, patterns, and files relevant to the PRD.

## Inputs
- Repo state.
- PRD reference.

## Outputs
- List of relevant files/components with short notes.
- Notes on patterns, conventions, and dependencies.

## Rules
- Do not propose changes yet.
- Use exact existing paths.

## Recommended model
- gpt-5.2-codex

## Suggested prompt
"Review the repo for files and patterns related to the PRD. Return a list of exact paths with short notes. Do not propose changes."
